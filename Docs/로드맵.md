# VeriHop 실행 로드맵 (자율 진행용)

작성: W1 진행 중. 목표: Baseline이 무너지는 멀티홉(현재 R@10 61.5%)을 **Ours(그래프 검증 Agentic RAG)**가 회복하는 것을 정량·데모로 증명.

이 문서는 남은 경로의 계획이다. 실제 진행 수치·결정은 `Docs/진행기록.md`에 append, 시각화는 대시보드에 갱신.

## 현재 상태 (기준선)
- ✅ W0.1 수율, W1.1 코퍼스(10,615), W1.2 FAISS(단일홉 R@10 98.5%)
- ✅ W1.5 Baseline: 단일홉 98.7% / 멀티홉 61.5% (−37p, 문제 증명)
- 🔄 W1.3 지식그래프: 초판 568노드/517엣지, 정밀도 63%(근거없는 사전지식 추출) → 근거문장 강제로 수정 중
- 🔄 W1.6 평가셋: 2Wiki 26문항(createdBy 편향), A(KorQuAD 정제) 병렬

## 자율 실행 순서 (오늘 밤)
각 단계는 background 완료가 다음을 트리거. 실패 시 원인 기록 후 다음 가능한 단계로.

### P1 — W1.3 그래프 완성 (정밀도 게이트)
- 근거문장 강제(evidence가 문단 substring이 아니면 폐기)로 재추출.
- **수용 기준**: 엣지 정밀도(solar-pro 근거 판정) ≥ 85%. 미달 시 프롬프트 1회 더 강화 후, 안 되면 현 수치 기록하고 진행(게이트 완화 사유 명시).
- 산출: `data/graph.pkl` (node[type], edge[relation, source_paragraph_id, evidence]).

### P2 — W1.4 alias 사전
- 코퍼스 doc_title + 2wiki ko title + (가능하면) 위키 redirect를 alias로.
- **수용 기준**: `data/alias.json` 생성, 2Wiki gold 엔티티(director·city)가 alias에 포함.
- 용도: name_check(후보∈alias) + 앵커 링킹.

### P3 — W2 도메인 순수 규칙 (mock 없이 테스트)
- `domain/plan_rules.py`(계획검증 8종), `domain/verify_rules.py`(Quad passed 산식, C1 veto).
- **수용 기준**: `tests/unit/` 통과. 시나리오 B 산술(빈 contradict→veto)이 verify_rules로 재현.

### P4 — W2 usecases + adapters (파이프라인 뼈대)
- adapters: `networkx_graph`(GraphPort, graph.pkl 로드), `chroma_index`→이미 FAISS(vector_store 어댑터), `json_tracer`.
- usecases: `correct`[0], `decompose`[1], `validate_plan`[2], `ground`[3], `resolve_hop`[4][5][6] 검색+검증, `executor`.
- `bootstrap.build_pipeline(mode)`.
- **수용 기준**: 2-hop E2E 1문항 통과(검색만→검색+검증). check_layers 게이트 통과.

### P5 — W3 검증·재질의 + W4 비교
- resolve_hop에 verifier 주입(quad/llm_judge) → run_eval basic/ours_g/ours 모드.
- **수용 기준**: 멀티홉에서 Ours R@10 > Baseline 61.5%. 3(4)단 비교표.

### P6 — 데모 (여유 시)
- Streamlit: 계획·hop trace·검증 뱃지·재질의·근거. 시나리오 A/B/C.

## 리스크
- **R1 그래프 커버리지**: 코퍼스 일부(552문단)만 추출 → 많은 엔티티 엣지 없음 → backlink weak_mode 빈발. 완화: eval 관련 문서 우선 추출, 전체 추출은 background 확대.
- **R2 평가셋 규모·편향**: 26문항 createdBy 편향 → 통계력 약함. 완화: A 합산, 스키마 확장(날짜·가족) 검토. 발표는 N 작음 캐빗 명시.
- **R3 LLM 추출/판정 노이즈**: 반복 확인됨(그라운딩·타입). 완화: 근거강제·타입가드·프롬프트 예시. 게이트로 정밀도 실측.
- **R4 데모 엔티티**: 모차르트/잘츠부르크가 코퍼스에 없으면 시나리오 B 재현 불가(M13). 완화: 해당 ko-위키 문서를 코퍼스에 추가.
